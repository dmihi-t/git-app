## 目指した課題解決

例えば一匹狼で生きてきたような強い人、または迷える子羊のように弱さをさらけ出している人、どんな人だって自身の生き方考え方に疑問を投げかけ、哲学する一瞬がある。その一瞬で消えてなくなってしまう貴重な哲学を、アウトプットし、共有する。そうすることで、全く別の世界への入り口へと誘われる。それは、狼が羊になる、または羊が狼になろうとするかのような、そんなきっかけを作りたい。
